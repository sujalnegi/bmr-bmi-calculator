# bmr-bmi-calculator
BMR (Basal Metabolic Rate) and BMI (Body Mass Index) calculator is a tool that takes your personal information to provide feedback on your daily diet and assess obesity status
![bmr-bmi-calculator](https://i.ibb.co/1f3rJCTx/Screenshot-2025-11-05-142616.png)
## Features
- TDEE Calculation: Total Daily Energy Expenditure using the Mifflin-St Jeor Equation
- BMI Calculation: Calculates Body Mass Index(BMI) and determines the corresponding health category (underweight, normal, overweight, obese).
- Basic Advice Generation: Provides a simple, one line piece of advice based on the user's BMI category.

## Technologies Used
- Flask Backend: A light weight flask server (app.py).
- Frontend: HTML5 and CSS
- JavaScript Backend: For background calculation

## Local Setup and Installation

follow these setps to get the application running on your local machine.

## 1. Prerequisites
- Python 3.7+
- `pip` (Python package installer)

## 2. Clone the Repositoy
Clone this repository to your local machine using Git:
```bash
git clone github.com/sujalnegi/bmr-bmi-calculator
cd bmr-bmi-calculator
```
## 3. Install Dependencies
Install all the required libraries
```bash
pip install flask
```

## 4. Run the application
Once the setup is complete, you can start the flask server
```bash
python app.py
```
Now, open your web browser and navigate to the following address:
```bash
http://127.0.0.1:5000
```
You should see the Face Extractor application running!

## How to Use
1. Open the link in the browser.
2. Enter your personal details.
3. Click on Calculate
4. A Result page will open which will tell your dialy calorie consuption.
5. And below it will the you BMI Index presented through a guage
